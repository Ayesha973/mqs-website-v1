# Mun quant society Web Client App

Frontend web application for Mun Quant Society.

Built using React, TypeScript, and Vite.

## Stack

- React
- TypeScript
- Vite
- ESLint
- Prettier

---

## Development

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build Project

```bash
npm run build
```

### Lint

```bash
npm run lint
```

### Format

```bash
npm run format
```
